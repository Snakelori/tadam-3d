ℹ️ Méthode de calcul (cliquer pour développer)
Coût produit réel = filaments + accessoires + logistique + fiche + électricité + usure + maintenance + main d'œuvre + mise en vente
Coût produit avec échec = Coût réel × (1 + taux d'échec %)
→ Intègre le risque d'impression ratée dans le coût unitaire
Coût fabrication = Coût produit × (1 + % charge vente)
→ Deux variantes : sans échec (coût réel) et avec échec
Prix de vente = Coût fabrication × taux de marge
→ Taux de marge défini dans Paramètres de calcul (ex : 3 = ×3 = marge 200%)
